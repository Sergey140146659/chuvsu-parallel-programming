# ChuvSU Parallel Programming

Репозиторий для заданий по параллельному программированию.

## Сборка task2.cpp

Для сборки и запуска `task2.cpp` используйте следующие команды:

```bash
g++-15 -O2 -std=c++17 -fopenmp task2.cpp -o task2
./task2
```
